# 🌐 alfhisa-web

**Muhammad Alfhi Saputra** — Lecturer & Researcher.
*Exploring Computer Science and sharing knowledge through teaching, research, and real-world projects.*

---

## ✨ Features

- **🏠 Home**: An overview of my work, research, projects, and teaching activities.
- **📚 Teaching**: Course materials and supporting resources used in my teaching.
- **🔬 Research**: Deep dives into Computer Science research and academic work.
- **🛠️ Projects**: Showcase of digital innovations and real-world builds.
- **✍️ Blog**: Articles and reflections on topics I explore and learn.
- **⚙️ Content Management**: Integrated with **Decap CMS** for easy markdown-based updates.

## 🚀 Tech Stack

- **Core**: Vanilla JavaScript (ES Modules)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Modern Vanilla CSS
- **CMS**: [Decap CMS](https://decapcms.org/)
- **Content**: Markdown-driven via `front-matter` & `marked`

### Course Objectives

1. Understand the DOM and browser architecture.
2. Build semantic and accessible HTML layouts.
3. Master modern CSS techniques (Flexbox, Grid, Variables).
4. Implement interactive features using Vanilla JavaScript.

### Syllabus Outline

- **Week 1-3:** HTML5 and CSS3 fundamentals. Building static layouts.
- **Week 4-6:** JavaScript basics, data types, DOM manipulation, and event handling.
- **Week 7-9:** Intermediate JavaScript (APIs, asynchronous programming, Promises).
- **Week 10-14:** Final project construction, deployment via Netlify/Vercel, and performance optimization.

### Grading System

*   30% — Homework and incremental assignments
*   20% — Midterm examination
*   50% — Final Website Project (Capstone)
